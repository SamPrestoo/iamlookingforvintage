📋 Setup Instructions

  1. Go to GitHub → Settings → Developer Settings → Personal Access Tokens
  2. Click "Generate new token (classic)"
  3. Give it a name like "iamlookingforvintage-admin"
  4. Select scope: repo (Full control of private repositories)
  5. Click "Generate token"
  6. Copy the token (starts with "ghp_") and paste it above
  7. Click "Save Token" and then "Test Connection"

Security: Your token is stored securely in your browser and never sent to anyone except GitHub.
